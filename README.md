# CODTECH Internship Tasks

This repository contains the tasks completed during my CODTECH internship.


## Task 1: Decision Tree Implementation

In this task, a Decision Tree model was implemented using Scikit-Learn to classify data from the Iris dataset.

**Steps performed:**
- Loaded the Iris dataset
- Split the dataset into training and testing sets
- Trained a Decision Tree Classifier
- Evaluated the model using accuracy score
- Visualized the decision tree structure

The model successfully predicts the class of iris flowers based on their features.


## Task 2: Sentiment Analysis with NLP

In this task, sentiment analysis was performed on customer reviews using Natural Language Processing techniques.

**Steps performed:**
- Created a dataset of customer reviews
- Applied TF-IDF vectorization to convert text into numerical features
- Split the dataset into training and testing sets
- Trained a Logistic Regression model
- Predicted sentiments of test reviews
- Evaluated the model using accuracy and classification report

The model successfully classifies reviews as positive or negative based on the textual data.

## Task 3: Image Classification using CNN

In this task, a Convolutional Neural Network (CNN) was built for image classification using TensorFlow.

**Steps performed:**
- Loaded the CIFAR-10 dataset
- Normalized image pixel values
- Visualized sample images from the dataset
- Built a CNN model using convolution and pooling layers
- Trained the model on training data
- Evaluated model performance on the test dataset

The CNN model successfully classified images into their respective categories.

## Task 4: Recommendation System

In this task, a recommendation system was developed using collaborative filtering techniques.

**Steps performed:**
- Loaded a sample user-item dataset
- Created a user-item interaction matrix
- Applied collaborative filtering techniques
- Generated personalized recommendations for users
- Evaluated the recommendation results

The system recommends items to users based on their preferences and similarity with other users.
